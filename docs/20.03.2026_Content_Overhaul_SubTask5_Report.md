# Content Overhaul — Sub-Task 5: New Exercise Types (Sentence Reorder + Fill-in-the-Blank Typing)

**Date:** 2026-03-21
**Branch:** `claude/calibrate-me-adaptive-learning-InrUV`
**Status:** Complete

---

## Objective

Add two new interactive exercise types beyond the existing multiple-choice and error-correction formats: **sentence-reorder** (arrange scrambled words) and **fill-in-the-blank typing** (type the answer). This diversifies the exercise portfolio, provides deeper cognitive engagement, and tests productive (not just receptive) language skills.

---

## What Was Done

### Step 1: Type System & Adapter Updates

- Extended `QuizItemType` union to include `'sentence-reorder' | 'fill-blank-typing'`
- Added `acceptableAnswers?: string[]` field to `QuizItem` for flexible answer matching
- Updated `OffGridActivity` interface and `adaptActivity()` to pass through new fields
- Added two new modules to `OFFGRID_MODULES` list

### Step 2: Created 25 Sentence-Reorder Exercises

Learners arrange scrambled words into correct English sentences. Targets areas where French word order differs:

| CEFR Level | Count | Focus Areas |
|------------|-------|-------------|
| A1 | 5 | Basic SVO order, adjective placement, negation with do-support |
| A2 | 3 | Question formation, adverb placement, indirect objects |
| B1 | 6 | Enough-placement, so...that, reported commands, exclamations, manner adverbs |
| B2 | 4 | Neither...nor, subject-auxiliary inversion, third conditional, relative clauses |
| C1 | 7 | Hardly...when, no sooner...than, inverted conditionals, under no circumstances, little did, seldom |

All exercises include:
- Scrambled words in `options` array
- Primary correct sentence + alternative acceptable orderings
- French comparison explaining word order differences
- Cameroonian context (Douala, Yaoundé, Bamenda, Kribi)

### Step 3: Created 25 Fill-in-the-Blank Typing Exercises

Learners type the missing word/phrase (no multiple choice). Targets productive skills:

| CEFR Level | Count | Focus Areas |
|------------|-------|-------------|
| A1-A2 | 8 | Irregular past tenses, prepositions (in/on/at), articles (a/an/the), basic collocations |
| B1-B2 | 10 | Phrasal verbs, since vs for, depend on, passive constructions, collocations |
| C1 | 7 | Future perfect, third conditional, inverted conditionals, gerund usage, bare infinitive, subject-auxiliary inversion |

All exercises include:
- Empty `options` array (no MCQ)
- `acceptableAnswers` with valid alternatives (spelling variants, contractions)
- Hint in parentheses within the question
- French comparison explaining common errors

### Step 4: Created Renderer Components

**SentenceReorder.tsx** — Interactive drag-to-build UI:
- Word bank of scrambled words
- Answer area where words are assembled
- Tap to add/remove words
- Submit button appears when all words are placed
- Correct/incorrect visual feedback with color coding
- Full feedback section with French comparison

**FillBlankTyping.tsx** — Text input UI:
- Auto-focused text input field
- Enter key to submit
- Case-insensitive + punctuation-normalized answer matching
- Correct/incorrect visual feedback on the input field
- Full feedback section with French comparison

### Step 5: Updated QuizContainer Routing

- New exercise types route through the same grammar flow (answer → confidence → feedback)
- Answer matching for new types uses case-insensitive comparison + `acceptableAnswers` checking
- Imports and conditional rendering added for both new components

### Step 6: Added CSS Styles

- `.sentence-reorder__*` classes for word bank, answer area, word chips, and states
- `.fill-blank__*` classes for input field and states
- Dark mode overrides for both components
