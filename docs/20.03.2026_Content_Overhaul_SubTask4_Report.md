## Objective

Expand grammar exercises from 329 to 500+ items, add three new C1-level modules (conditionals, passive voice, reported speech), upgrade all 3-option MCQ exercises to 4 options with plausible French-L1 distractors, and rebalance the CEFR/difficulty distribution across all modules.

---

## What Was Done

### Step 1: Added 60 C1 Grammar Exercises (3 new modules)

Three new advanced grammar modules targeting C1 learners, all with Cameroonian context and French comparison fields:

| Module | Items | Topics |
|--------|-------|--------|
| Conditionals | 30 | Mixed/third/inverted conditionals, wish, if only, unless |
| Passive Voice | 25 | Reporting passives, modal perfect passive, perfect participle passive |
| Reported Speech | 20 | Backshift, subjunctive after suggest/recommend/insist, say vs tell |

These fill the C1 gap identified in Sub-Task 1 (previously 0 C1 grammar exercises).

### Step 2: Expanded Existing Modules (249 → 374 exercises)

Each existing module was expanded with new exercises at varied CEFR levels:

| Module | Before | After | Added |
|--------|--------|-------|-------|
| Tense & Form | 50 | 70 | +20 |
| Subject-Verb Agreement | 40 | 50 | +10 |
| Articles | 20 | 40 | +20 |

| Prepositions | 30 | 50 | +20 |

| Cameroonian Scenarios | — | 69 | (unchanged) |
| False Cognates | — | 50 | (unchanged) |

### Step 3: Upgraded All Exercises to 4 Options

All 286 original 3-option MCQ exercises were upgraded to 4 options. The 4th distractor was designed using French L1 interference patterns:

- **Grammatically plausible** but incorrect forms that French speakers would produce
- **Common L1 transfer errors** (e.g., article overgeneralization, preposition calques)
- **Near-miss distractors** that test fine-grained knowledge

Result: 494/509 exercises now have 4 options. The remaining 15 are error-correction type (no MCQ options).

### Step 4: Fixed `frenchComparison` Format

Standardized the `frenchComparison` field across all exercises to use the structured format:

```typescript
frenchComparison: {
  frenchStructure: string;
  englishStructure: string;
  whyDifficult: string;
  visualHighlighting?: { incorrect: string; correct: string };
}
```

### Step 5: Registered New Modules

Added three new entries to `OFFGRID_MODULES` in `src/data/offgridAdapter.ts`:

```typescript
{ id: 'conditionals', name: 'Conditionals', count: 30 },
{ id: 'passive-voice', name: 'Passive Voice', count: 25 },
{ id: 'reported-speech', name: 'Reported Speech', count: 20 },
```

### Step 6: Rebalanced CEFR/Difficulty Distribution

Remapped difficulty values to ensure proper CEFR correlation across the expanded exercise set.
