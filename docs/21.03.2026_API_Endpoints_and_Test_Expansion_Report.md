## Part 1: Backend API Endpoints (Commit 3e7c61c)

### Problem

The frontend `aiService.ts` defined 9 API methods, but only 4 had corresponding Vercel serverless endpoints. Five methods threw "not implemented" errors at runtime:

| Missing Endpoint | Purpose |
|---|---|
| `explainVocabulary` | AI-powered vocabulary word explanations with etymology and usage |
| `generateGrammarExercises` | Dynamic grammar exercise generation by topic |
| `generateProgressReport` | Comprehensive learning progress analysis |
| `generateStudyPlan` | Personalized study plan generation |
| `getPronunciationGuide` | IPA transcription and pronunciation tips |

### Implementation

**5 new Vercel serverless functions** were created in `api/`:

| File | Lines | Model | Description |
|---|---|---|---|
| `api/explain-vocabulary.ts` | 53 | `llama-3.3-70b-versatile` | Returns etymology, definition, example sentences, common mistakes, French-speaker tips |
| `api/generate-grammar-exercises.ts` | 73 | `llama-3.3-70b-versatile` | Generates exercises with options, correct answers, and explanations by topic/count |
| `api/generate-progress-report.ts` | 68 | `llama-3.3-70b-versatile` | Analyzes stats to produce strengths, weaknesses, recommendations, overall assessment |
| `api/generate-study-plan.ts` | 80 | `llama-3.3-70b-versatile` | Creates multi-day study plans with daily activities, focus areas, and weekly goals |
| `api/pronunciation-guide.ts` | 59 | `llama-3.3-70b-versatile` | Provides IPA transcription, syllable breakdown, audio description, common errors |

**Shared type definitions** were extracted into `api/_types.ts` (127 lines) to ensure consistent request/response typing across all endpoints.

### Frontend Integration

**Modified files:**

- `src/features/quiz/components/QuizContainer.tsx` (+11 lines) — integrated vocabulary explanation trigger
