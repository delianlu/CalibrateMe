## Objective

Expand vocabulary packs to publish-ready volume (Essential: 150, Academic: 150, Business: 100) and add `frenchTrap` fields to ~25-30% of all items to flag common pitfalls for French-speaking learners.

## What Was Done

### Step 1: Added FrenchTrap type to QuizItem interface

Added new types in `src/features/quiz/types.ts`:

```typescript
export type FrenchTrapType = 'false-friend' | 'pronunciation' | 'gender-confusion' | 'spelling' | 'usage-difference' | 'none';

export interface FrenchTrap {
  type: FrenchTrapType;
  warning: string;
  correctUsage: string;
}
```

Added `frenchTrap?: FrenchTrap` field to the `QuizItem` interface.
