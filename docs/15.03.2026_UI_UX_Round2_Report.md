## Fixes (5)

### 1. Simulation Lab Button Hierarchy
**File:** `src/components/SimulationControls.tsx`, `src/App.css`

- Introduced three distinct button tiers: **primary** (gradient, glowing), **secondary** (outlined, translucent), **ghost** (text-only, subtle)
- "Run Simulation" uses `.btn-primary`, parameter toggles use `.btn-secondary`, less important actions use `.btn-ghost`
- Hover/active states for each tier with appropriate visual weight

### 2. Practice Mode Selector — 3-Column Grid
**File:** `src/features/quiz/components/QuizContainer.tsx`, `src/App.css`

- Replaced the inline button row with a 3-column CSS grid layout
- Each mode card now has an **icon above text** layout (emoji icon + label + description)
- Active mode highlighted with neon accent border and subtle glow
- Cards use consistent padding, border-radius, and dark glass-morphism background

### 3. Confidence Slider — Gradient Track + Tick Marks
**File:** `src/features/quiz/components/ConfidenceSlider.tsx`, `src/App.css`

- Slider track now displays a **gradient** from red (0%) through yellow (50%) to green (100%)
- Added **tick marks** at 0%, 25%, 50%, 75%, and 100% positions
- Endpoint labels: "Guessing" (left) and "Certain" (right) in muted text
- Custom thumb styling with glow effect on hover/active

### 4. Session Summary — ProgressRing + Stacked Bar + Action Buttons
**File:** `src/features/quiz/components/SessionSummary.tsx`, `src/App.css`

- **ProgressRing:** SVG donut chart showing accuracy percentage with animated stroke-dashoffset, color-coded (green/yellow/red based on score)
- **Dual-process stacked bar:** Horizontal bar visualizing Fast (System 1) vs Slow (System 2) response distribution with distinct gradient fills
- **Action buttons:** "Review Mistakes" and "Next Session" with primary/secondary styling
- Stats displayed in a responsive 2×2 grid with icon + value + label layout

### 5. Vocabulary Cards — Mastery Borders + Progress Bar + Review Date
**File:** `src/features/vocabulary/components/VocabularyCard.tsx`, `src/App.css`

- Each card now has a **colored left border** reflecting mastery level:
  - New → gray, Learning → blue, Reviewing → amber, Mastered → green
- **Progress bar** beneath each card showing mastery percentage with matching color
- **Next review date** displayed in small muted text with calendar icon
- Hover lift effect with subtle shadow transition

## Improvements (7)

### 1. Sidebar Collapse Toggle
**File:** `src/components/Dashboard.tsx`, `src/App.css`

- Added a collapse/expand button (`☰` / `✕`) at the top of the sidebar
- Collapsed state persisted to **localStorage** (`sidebar-collapsed` key)
- Sidebar animates between full width (240px) and icon-only width (56px)
- Navigation labels hidden in collapsed state; icons remain visible
- Main content area smoothly expands to fill available space

### 2. Quick Stats Bar on Practice Tab
**File:** `src/components/Dashboard.tsx`, `src/App.css`

- Horizontal stats strip displayed above the practice content area
- Shows 4 key metrics: **Due Today**, **Streak**, **ECE Score**, **XP**
- Uses monospace font for values, muted labels below each metric
- Glass-morphism background consistent with card styling

### 3. Page Transitions Refinement
**File:** `src/App.css`

- Easing curve updated from default to `cubic-bezier(0.4, 0, 0.2, 1)` to match Material-style motion spec
- Applied to tab content transitions, sidebar expand/collapse, and card hover effects
- Consistent 200ms duration across all micro-interactions

### 4. Custom ChartTooltip Component
**Files:** `src/components/ChartTooltip.tsx` (new), + 7 chart files

- Created a **reusable custom Recharts tooltip** replacing all 15 default `<Tooltip />` instances across the codebase
- Styled with dark glass-morphism background, colored dots per series, and formatted values
- Accepts an optional `formatter` prop for value formatting (e.g., `.toFixed(3)`, `%` suffix)
- Applied to: `CalibrationChart`, `ComparisonView`, `DomainCalibrationChart`, `CrammerCrashChart`, `MasteryComparison`, `DoseResponseChart`, `LearnerClassification`, `FinalReport`

### 5. Loading Skeletons
**File:** `src/components/Skeleton.tsx` (new), `src/App.css`

- Created a reusable **skeleton loading system** with 6 exports:
  - `Skeleton` — base shimmer block (configurable width, height, border-radius)
  - `SkeletonCard` — card placeholder (title + value + subtitle)
  - `SkeletonGrid` — configurable grid of skeleton cards
  - `AnalyticsSkeleton` — 4-card row + 2 chart placeholders
  - `VocabularySkeleton` — search bar + 3×3 card grid
  - `ProfileSkeleton` — header + progress bar + 3-card row
- CSS shimmer animation using gradient keyframe
- All skeletons include `aria-hidden="true"` or `aria-label="Loading..."` for accessibility

### 6. Simulation Lab Results Polish
**File:** `src/features/analytics/components/CalibrationDashboard.tsx`, `src/App.css`

- Added **profile badge** and **scheduler badge** chips to simulation results
- Introduced a **tabbed sub-navigation** within the results panel (Overview / Charts / Data)
- Badge styling uses pill shape with color-coded backgrounds per learner profile

### 7. Footer Refinement
**File:** `src/App.tsx`, `src/App.css`

- Footer size reduced (smaller font, tighter padding)
- Opacity lowered to 0.5, rising to 0.7 on hover
- Footer **hidden entirely** on Simulation Lab and Analytics tabs to maximize content area
- Smooth fade transition when appearing/disappearing between tabs
